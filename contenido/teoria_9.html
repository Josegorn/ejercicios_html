Formularios

La primera vez que te pongas a leer pensarás que son decenas de etiquetas nuevas, pero pronto te darás cuenta de que se repiten muchas con pequeñas variaciones. !Ánimo!

Para comenzar el formulario usamos la etiqueta FORM y se cerrará como siempre al terminarlo.

  <form action:”mailto:dirección de correo” method=”post” nctype=”text/plain”>

Dentro de un formulario podemos usar distintos campos del formulario vamos a ver los más habituales:

   <input type=”text” name=”nombre del campo que guardara la información”> Indica recuadro para recoger información. Soporta los siguientes modificadores

  size=”xxx” Longitud del recuadro en caracteres.
  maxlength=”xxx” Numero máximo de caracteres permitidos

   <input type=”password” name=”nombre del campo que guarda la información”> Indica recuadro para recoger información, pero esta información saldra cifrada. Soporta los siguientes modificadores

  size=”xxx” Longitud del recuadro en caracteres.
  maxlength=”xxx” Numero máximo de caracteres permitidos

   <textarea name=”nombre del campo que guardara la información rows=”número de filas” cols=”número de columnas”> Texto que contendra el recuadro de texto por defecto </textarea> Indica que se va a crear un recuadro de texto para almacenar gran cantidad de información

   <select name=”nombre del campo que guardara la información”>  Opciones </select> Soporta los siguientes modificadores:

  <option> Indica las distintas opciones que tendrá el usuario para elegir se usa una de estas etiquetas por cada opción.
  <múltiple size=”número de opciones que el usuario puede ver”> Indica que se puedan ver varias opciones a la vez. Es un modificador de select

   <input type=”checkbox”  name=””nombre del campo que guardara la información”> Indica un campo que podrá ser seleccionado o no. Puedes usar el siguiente modificador:

  checked Modificador de input que indica que estará marcado por defecto.

   <input type=”radio”  name=””nombre del campo que guardara la información” value=”valor que almacenara dicho campo”> Indica que se va a dar a escoger entre  varias opciones a un usuario pero en forma de botones circulares. Cada una de las opciones se indica con:

  checked Modificador de input que indica que estará marcado por defecto.

   <input type=”submit”  value=”valor que mostrara el botón”> Indica la presencia de un botón para enviar el formulario

  <input type=”reset”  value=”valor que  mostrara el botón”> Indica la presencia de un botón para borrar todos los campos del formulario

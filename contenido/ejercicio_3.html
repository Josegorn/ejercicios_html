LISTAS EN PAGINAS WEBS

Bienvenido a la zona  de practicas para listas.

Aquí te familiarizaras con los tres tipos de listas mas usados en el formato HTML

Empecemos con las listas ordenadas. Que son aquellas que están numeradas. Es decir cada línea tiene un numero único asociado

  1. Para crear una lista la lista debe comenzarse con la etiqueta OL si es ordenada.

  2. Después se van enumerando cada uno de los elementos de las listas con la etiqueta LI. Es decir cada uno de estos párrafos que ves numerados empezarian con un LI.

  3. Finalmente la lista se cierra con la etiqueta de cierre OL.

 

Los pasos a seguir con las listas no ordenadas son idénticos a las listas ordenadas con la salvedad de que la lista se abre y cierra con la etiqueta UL. Es decir

 

  • Para crear una lista la lista debe comenzarse con la etiqueta UL si es ordenada.

  • Después se van marcando cada uno de los elementos de las listas con la etiqueta LI. Es decir cada uno de estos párrafos que ves numerados empezariín con un LI.

  • Finalmente la lista se cierra con la etiqueta de cierre UL.

Las listas de definición son aquellas en las que el primer párrafo de lista es un termino a definir y en el siguiente se define el termino y así sucesivamente. Como esto es difícil de explicar aquí va un ejemplo.

            BR

                        Se utiliza para separar párrafos

            HR

                        Se utiliza para insertar separadores horizontales

            YR

                        No tiene uso es una errata :P

Bueno si has llegado hasta aquí es porque lo has entendido todo perfectamente.

Bien ahora toca la ultima parte quizás algo mas complicada el anidamiento de listas.

Decimos que una lista esta anidada cuando esta dentro de esta lista, por ejemplo

Podrías enumerar tus aficiones como sigue:

Sin un orden particular, mis aficiones son las siguientes:

MIS AFICIONES

  • El cine

  • El deporte

    • Natación

    • Baloncesto

  • La música

Incluso se podrían anidar listas como

MIS AFICIONES

  • El cine
     
  • El deporte
    1. Natación

    2. Baloncesto
  • La música
